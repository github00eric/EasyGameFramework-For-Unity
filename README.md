# EasyGameFramework

## 介绍

用于快速原型游戏制作的游戏引擎中间代码框架。开发策略是易用度优先。

- 包含功能：
- 解耦：
    - IOC容器 EGFEntry
    - 事件系统
    - 命令
    - 绑定数据
- 模板：
    - 单例模板
    - UI页面模板
- 工具：
    - 日志工具
    - 对象池
    - （Editor）代码修改项目宏定义
    - （开源）JCMG Nodey，一个泛用可视化图形节点框架
    - （付费）Odin
- 模块：
    - 音频播放模块
    - 资源加载模块
    - UI基本框架
